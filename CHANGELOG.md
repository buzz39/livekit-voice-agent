# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-05

### Added
- Initial production release
- LiveKit telephony agent with AI voice capabilities
- n8n MCP integration for workflow automation
- Support for GPT-4o-mini, Deepgram Nova-3, Cartesia Sonic-2
- Docker deployment support
- Comprehensive documentation
- Example environment configuration
- Startup scripts for Windows and Unix systems
- Outbound call utility script

### Features
- Natural language phone conversations
- Real-time speech recognition and synthesis
- Voice activity detection
- Dynamic tool loading from n8n
- Extensible function tool system
- Production-ready error handling and logging
- Time-based personalized greetings

### Technical
- Python 3.12+ support
- MCP (Model Context Protocol) integration
- SSE-based persistent connection to n8n
- Optimized for telephony audio quality
